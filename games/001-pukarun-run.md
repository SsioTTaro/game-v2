# Game #001：ぷかるんラン

## ステータス

**ドラフト** — 2026年5月7日 / プラットフォーム第1作 / 6月公開目標

> プラットフォーム共通仕様は `../platform-design.md` を参照。本ドキュメントは個別ゲーム固有の仕様のみを記述する。

---

## 1. 概要

| 項目 | 内容 |
|------|------|
| game_id | `endless_run` |
| ジャンル | エンドレスラン × ハイスコア |
| 想定セッション | 1〜2分 |
| 対象端末 | スマホ主、PC両対応 |
| 操作 | 1ボタン（タップ／クリック／スペース） |

### コアの一文

> **ぷかるんを操作して障害物を避け続け、運次第で大化けするスコアを狙う、ゲームセンター感覚のスルメゲー。**

---

## 2. ゲームルール

- ぷかるん（風呂敷キャラ）が画面左に固定、背景が右から左に流れる
- 地面に障害物が出現、ジャンプで避ける
- 当たるとゲームオーバー → Result 画面へ
- スクロール速度は時間経過で徐々に上昇

### 操作

| 端末 | 操作 |
|------|------|
| スマホ | タップ＝ジャンプ、長押し＝大ジャンプ |
| PC | スペース or マウスクリック＝ジャンプ |

レーン変更・二段ジャンプは**入れない**（シンプルさ優先）。

---

## 3. スコア計算

| 要素 | 加点 |
|------|------|
| 進行距離 | 1距離 = 1点 |
| 障害物すれすれ回避（コンボ） | 連続成功で倍率上昇（最大 ×3） |
| レアアイテム取得 | 第4節参照 |

Result 画面でスコア内訳を表示（基礎点・コンボボーナス・アイテム効果）。

---

## 4. 運要素（脳汁ポイント）

スコアが大化けする仕掛け。低確率発生を意図的に作り、平均から大きく外れた値を狙えるようにする。

| アイテム | 出現率 | 効果 | キャラとの関連 |
|----------|--------|------|----------------|
| メロンソーダ缶 | 中（数十秒に1回） | 取得後一定時間スコア倍率 ×2 | るるの好物 |
| 桜の花びら | 低（数分に1回） | 一定時間ぷかるん浮遊、障害物無視 | 春の髪飾りモチーフ |
| クラシックギター | 超低 | 紗倉るるが歌い始め、BGM切替＋倍率 ×3 | るるの好物（クラギ） |

---

## 5. 演出・ニヤけ要素

すべて言語依存のテキストは紗倉るる側に集中、**ぷかるんは鳴き声＋モーションのみ**。

### 5-1. 紗倉るるの応援

| トリガー | 内容 |
|----------|------|
| ゲーム開始時 | 「いってらっしゃい」系セリフ |
| スコア帯到達（500/1000/3000等） | ベスト更新煽り |
| ゲームオーバー | スコア帯ごとのリアクション |

時間帯（昼/夜）はゲーム開始時にランダム選定し、応援トーンが変わる（昼＝元気／夜＝おっとり）。

### 5-2. ぷかるんのリアクション

- 失敗時：転がり方バリエーション複数
- 浮遊中：ぷかぷか上下する慣性的な動き
- スコア更新時：飛び跳ねる

### 5-3. 隠し要素

- 隠しスコア（1234, 7777, 等）到達で特殊演出
- 連続プレイ N 回目で初めて出るレア演出

## 7. データ：`metadata` の中身

`scores.metadata` に格納するゲーム固有情報（JSONB）：

```json
{
  "max_combo": 12,
  "items_collected": {
    "melon_soda": 3,
    "sakura": 1,
    "guitar": 0
  },
  "play_duration_sec": 87,
  "play_time_of_day": "day"
}
```

ランキング表示時、上位プレイの内訳を見せるのに使う（オプション機能）。

## 8. アセット要件（仮素材で進行、後で差し替え）

| 種別 | 必要なもの | 仮素材方針 |
|------|------------|------------|
| ぷかるん | 待機・ジャンプ・落下・転倒モーション、浮遊モーション、鳴き声SE | 仮：シンプル図形（緑の丸＋耳）、フリー素材SE |
| 紗倉るる | 立ち絵（昼／夜）、応援セリフ吹き出し | 仮：シルエット表示、テキストのみ |
| 背景 | スクロール対応の地面・空・遠景 | 仮：単色グラデーション＋シルエット |
| 障害物 | 数種類（小・中・連続配置パターン） | 仮：幾何学図形 |
| アイテム | メロンソーダ缶、桜の花びら、クラシックギターの3種 | 仮：絵文字・図形 |
| BGM | 1曲（通常）＋レアイベント時切替 | フリー素材／AI生成 |
| SE | ジャンプ、着地、アイテム取得、ゲームオーバー、ハイスコア更新 | フリー素材 |
